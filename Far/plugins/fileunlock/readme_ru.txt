Плагин "File unlock" для Far Manager 3.0
****************************************

Плагин, позволяющий принудительно закрывать файлы и каталоги
(в т.ч. в режиме рекурсивного поиска), используемые другими
приложениями.

Ограничения:
  1. Файлы-модули (библиотеки dll), загруженные приложениями посредством
     вызова LoadLibrary доступны только для просмотра и не могут быть
     закрыты принудительно.
  2. Far должен быть запущен с правами пользователя, не ниже чем процесс,
     открывший файл. Другими словами - если файл открыт под администратором,
     а плагин вызван из под обычного пользователя - дескриптор не будет
     найден.
  3. Нельзя отследить файлы, используемые системными сервисами или драйверами.
  4. Операционная система: MS Windows Vista/7 или MS Windows Server 2008.


Известные баги:
  После использования плагина возможно "зависание" Far при выходе.

Установка:
  Распакуйте содержимое архива в каталог плагинов Far (...Far\Plugins).

Хозяйке на заметку:
