Плагин "JClassInfo" для Far Manager 3.0
***************************************

Просмотр и декомпиляция Java классов.
Декомпиляция производится с помощью Fernflower (F4), JAD (F3) или Javap (F5).

Установка:
  Распакуйте содержимое архива в каталог плагинов Far (...Far\Plugins).

Хозяйке на заметку:

  ответственности за последствия использования данного плагина.
