Плагин "PEAnalyzer" для Far Manager 3.0
***************************************

Анализатор исполняемых файлов MS Windows (x86/x64) со встроенным дизассемблером.
Дизассемблер основан на библиотеке BeaEngine (http://www.beaengine.org/).

Возможности текущей версии:
  - Просмотр таблиц импорта/экспорта;
  - Просмотр ресурсов;
  - Дизассемблер (i386/amd64);
  - Просмотр информации о заголовках DOS/NT и секциях PE файлов;
  - Определение сборщика/упаковщика по сигнатуре.

Установка:
  Распакуйте содержимое архива в каталог плагинов Far (...Far\Plugins).
