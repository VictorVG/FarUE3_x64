Плагин "AudioPlayer" для Far Manager 3.0
****************************************

Простой аудио плеер на основе библиотеки BASS (http://www.un4seen.com/).

Поддерживаются следующие форматы файлов:
  *.aiff,*.mp1,*.mp2,*.mp3,[*.mp4],*.ogg,*.wav,*.mo3,*.it,*.xm,*.s3m,*.mtm,
  *.mod,*.umx,*.m4a,*.flac,*.fla,*.oga,*.wma,*.wv,*.wvc,*.aac,*.m4b,*.ac3,
  *.ape,*.mac,*.mpc,*.mp,*.mpp,*.spx,*.midi,*.mid,*rmi,*.kar

Установка:
  Распакуйте содержимое архива в каталог плагинов Far (...Far\Plugins).

Хозяйке на заметку:
  Данный плагин предоставляется "as is" ("как есть"). Автор не несет
  ответственности за последствия использования данного плагина.
