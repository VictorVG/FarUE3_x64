GitSHell for FAR 3
------------------

Диалоговый интерфейс к системе контроля версий GIT

Плагин предназначен для упрощения работы с системой контроля версий GIT. 
Поддерживается базовый набор действий: checkout, commit и просмотр истории. 
Все что нельзя сделать с помощью плагина - делайте через командную строку GIT.

Для сравнения веток/файлов требуется наличие плагина Visual Compare ver 18+.

-------------------------------------------------------------------------------
История изменений:

Ver 1.1

- Диалог Commit, позволяет задать подробное описание Commit'а и автора.
  Так же позволяет изменить последний commit (Amend last commit)

- В списке истории поддерживается выделение (Ins, Ctrl+, Ctrl-)

- В списке истории по Alt-F3 можно сравнить 2 выделенных Commit'а

- Возможность сравнения merge commit

- В списке истории поддерживается показ сокращенных ID

- Командная строка плагина:

  git:  
    Главное меню плагина

  git:branch 
    Список веток

  git:log {<Каталог/Файл>}
    История изменения

  git:commit {<Каталог/Файл>}
    Окно фиксации (commit)

- Исправлено несколько ошибок

Ver 1.2

- Исправлена ошибка в x64 версии
