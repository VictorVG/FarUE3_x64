Плагин "TextConv" для Far Manager 3.0
*************************************

Обработка текстовых файлов:
  - удаление лишних концевых пробелов и знаков табуляции;
  - удаление пустых строк из конца файла;
  - преобразование кодов концов строк (CRLF<->LF<->CR);
  - перекодировка (ANSI<->UTF8<->...);
  - замена спецсимволов на пробелы;
  - замена пробелов на табуляцию;
  - замена по регулярным выражениям на базе PCRE.

Не поддерживаются (пока):
  - обратные ссылки в регулярных выражениях.

Установка:
  Распакуйте содержимое архива в каталог плагинов Far (...Far\Plugins).

Использование:
  В режиме редактора: F11 -> Преобразование текста.
  В режиме панелей: отметить файлы для обработки -> F11 -> Преобразование текста.
