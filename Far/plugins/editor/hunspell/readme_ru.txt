Плагин "Hunspell" для Far Manager 3.0
*************************************

Проверка орфографии во встроенном редакторе Far.

Базируется на библиотеке hunspell (http://hunspell.github.io/).
В поставку входят следующие словари:
  - Английский;
  - Русский.

Установка:
  Распакуйте содержимое архива в каталог плагинов Far (...Far\Plugins).

Использование:
  1. Откройте текстовый файл во встроенном редакторе Far;
  2. Нажмите F11 -> Проверка орфографии. Проверка начинается с текущего
     положения курсора.
