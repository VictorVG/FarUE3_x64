Плагин "PortaDev" для Far Manager 3.0
*************************************

Обёртка над Windows Portable Devices API, позволяющая работать
с накопителями устройств как с обычной файловой системой по
протоколу MTP (Media Transfer Protocol).

Установка:
  Распакуйте содержимое архива в каталог плагинов Far (...Far\Plugins).

Хозяйке на заметку:

  ответственности за последствия использования данного плагина.
