Плагин "Media Info" для Far Manager 3.0
***************************************

Плагин позволяет просматривать информацию о медиафайлах (кодек,
битрейт, продолжительность, EXIF и т.д.).

Для чтения информации используются библиотеки:
- MediaInfo (http://mediainfo.sourceforge.net);
- Exiv2 (http://www.exiv2.org);
- Стандартные средства GDI+ (http://www.microsoft.com).

Установка:
  Распакуйте содержимое архива в каталог плагинов Far (...Far\Plugins).
