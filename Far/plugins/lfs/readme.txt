      LuaFAR Search v3.x
====================================

*  This is a plugin for Far Manager 3.0.
*  It works in Far Manager editor and panels.
*  How to use: see the help file.
*  The plugin is written in Lua programming language.

Installation
-------------
1. Create a directory under %FARHOME%\Plugins, extract files
   and subdirectories into it, then restart Far.

Note
-----
This package works with Far Manager regular expressions library.
In order to add more libraries (Oniguruma, PCRE, PCRE2),
install "LuaFAR Search add-on" (a separate download).
