
Ver 1.0.1,  06.12.2007

 + Добавлена возможность форсирования вызова хинта при нажатии префиксной 
   клавиши (по умолчанию - Shift, настраивается через Settings.ini).
   Работает при выключенном автотрекинге мыши. Кроме того, пока удерживается
   префиксная клавиша хинт не исчезает при перемещении мыши - удобно для
   поиска картинок и т. п.

 * Исправлена ошибка с нестабильностью ручного вызова хинта через макрос

 * Изменен файл Setup.reg. Теперь для вызова плагина предлагается использовать
   макросы AltShift1 и AltShift2, это позволяет избежать исчезания окна
   быстрого поиска при включенном автотрекинге клавиатуры.

Ver 1.0.2,  09.12.2007

 * Исправлена ошибка, приводившая к повышенной нагрузке процессора при запуске
   из фара внешних программ и ряд других ошибок.

 + Добавлена совместимость с эмулятором консоли ConEmu.

 + Добавлен плагин для показа MP3 тэгов. Пока показываются только тэги v1.

 + Добавлен новый плагин для показа анимированных курсоров. Данный плагин
   написан в демонстрационных целях, и включен вместе в исходными текстами.
   Исходные тексты плагина и Delphi версия FarHints API - в каталоге Sources.

Ver 1.0.3,  11.12.2007

 + Плагин Image теперь показывает анимированные изображения

 * Исправлено несколько ошибок.

Ver 1.0.4,  16.12.2007

 + Добавлена возможность управлять прозрачностью хинта (ключ Transparency)

 + Теперь хинты могут показываться не только для файловых панелей, но и для
   панелей плагинов. По умолчанию показываются только стандартные свойства:
   Name, Description, Modified, Size. Возможно создание субплагинов FarHints
   для показа "особых" свойств плагинов Far'а

 + Добавлен субплагин, работающий в панели плагина ProcessList, показывающий
   VersionInfo и иконку процесса.

 + Длинные строки (например, Description) теперь сворачиваются в несколько строк.

 * Изменения в FarHints API для поддержки субплагинов для плагинов :)

Ver 1.0.5,  17.12.2007

 * Исправлена ошибка версии 1.0.4, приводившая к падению Far'а под ConMan'ом.

Ver 1.0.6,  21.12.2007

 + Добавлен новый режим вывода хинта - "Permanent hint". При включении этого
   режима появляется хинт для текущей позиции, который не исчезает при 
   перемещении между файлами. Режим включается/выключается через меню 
   плагина

 + Улучшена работа под консольным менеджером ConMan. Теперь версии Far'а
   на неактивных страницах не пытаются показать хинт. Тестировалось под
   ConMan v 2.33

 + Плагин теперь работает в контекстах Quick View и Tree Panel (т.е. хинт
   от мыши будет появляться на пассивной панели). Для этого нужно зарегистрировать
   дополнительные макросы из SetupAdd.reg

Ver 1.0.7,  25.12.2007   

 + Добавлена поддержка Shell Thumbnail (работает только под Windows Vista).
   Для тех типов документов, для которых существует ThumbnailProvider, в хинте
   отображается эскиз документа, такой же как и в Explorer'е.  Фичи:

   - Эскиз извлекается в отдельном потоке и кэшируется (используется системный
     thumbnail cache).

   - Размеры эскизов настраиваются отдельно для папок и остальных документов
     (ключи MaxThumbnailSize, MaxFolderThumbnailSize)

   - Поверх эскиза показывается иконка документа (ключ IconOnThumbnail)

Ver 1.0.8,  27.12.2007

 + Добавлена поддержка Shell Thumbnail для Windows XP.

 + Во время извлечения эскиза иконка документа моргает

 + Частичная поддержка деревьев. В контексте дерева хинт работает только 
   для текущего каталога (не поддерживается автотрекинг мыши). 
   Для работы автотрекинга клавиатуры нужно зарегистрировать соответствующий
   макрос из SetupAdd.reg

Ver 1.0.10

 + Добавлена возможность интерактивного масштабирования эскизов.
   Для использования этой фичи необходимо назначить макросы, вызывающие
   соответствующие команды из меню плагина. Вызывать эти команды
   непосредственно из меню нельзя (так как при открытии меню хинт 
   исчезает)

   В файле SetupWheel.reg приведены примеры таких макросов, повешенные
   на колесико мыши и кнопки "+" / "-" (на цифровой клавиатуре)

 + Улучшен алгоритм извлечения Shell Thumbnail под Windows Vista, 
   теперь почти всегда извлекаются эскизы желаемого размера.

 + Субплагин Image также поддерживает интерактивное масштабирование,
   максимальный размер эскиза ограничен оригинальным размером картинки.
   Рендеринг больших картинок выполняется в отдельном потоке, в это
   время показывается preview картинки (если есть).

 * Субплагин VerInfo показывает информацию даже в случае если структура
   файла имеет некоторые нарушения.

 * Изменено главное меню плагина, некоторые опции вынесены в отдельное
   подменю, при изменении они сохраняются в реестре

 + Изменен цвет хинта по умолчанию (для соответствия Windows Vista),
   добавлена возможность градиентной закраски хинта (настраивается
   через файл Settings.reg)

 * Изменения в FarHints API для поддержки команд масштабирования.

 * Юникодная версия настроена на новый build Far2
