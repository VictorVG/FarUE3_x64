Плагин "Sudo" для Far Manager 3.0
*********************************

Плагин, позволяющий запускать процессы с повышенными/пониженными привилегиями.
Аналог команды "sudo"/"su" в linux. Актуален для систем MS Vista/Seven с
включённым UAC.

Использование:
  Плагин обрабатывает следующие префиксы:

  - Запуск процесса с повышенными привилегиями:
    sudo:{Приложение} [Параметры]

  - Запуск нового экземпляра консоли с повышенными привилегиями и выполнение
    указанной команды. Консоль после выполнения команды не закрывается:
    su:[Команда [Параметры]]

  - Запуск процесса с пониженными привилегиями:
    sudown:{Приложение} [Параметры]

Установка:
  Распакуйте содержимое архива в каталог плагинов Far (...Far\Plugins).
