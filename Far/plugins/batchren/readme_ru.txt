Плагин "Batch rename files" для Far Manager 3.0
-----------------------------------------------

Плагин предназначен для пакетного переименования группы файлов, выделенных
на файловой панели. Для изменения имен используется редактор Far Manager.

Для переименования старое и новое имена должны быть заключены в кавычки и находиться в одной строке.
Например, переименование файла filename_old.txt в filename_new.txt:
"filename_old.txt" "filename_new.txt"

Плагин позволяет переносить файлы в другой каталог.
Например, перенос файла filename.txt из текущей панели в корень диска c:
"filename.txt" "c:\filename.txt"

Плагин поддерживает работу с переменными окружения.
Например, перенос файла filename.txt во временный каталог:
"filename.txt" "%temp%\filename.txt"

Есть возможность включать в список для переименования файлы из подкаталогов.
Для этого необходимо переименовать BatchRen.dll.example.ini в BatchRen.dll.ini
(в меню плагинов появится пункт "Batch rename incl. subfolders").
Внимание! В списке файлов с подкаталогами имена каталогов идут раньше имен
файлов, поэтому возможна ситуация, когда при переименовании каталога путь к
остальным файлам изменяется и система не может их найти.

В BatchRen.dll.example.ini с помощью TitleFKey можно выключить или
перенести информацию о режиме копирования/переименования в панели
функциональных клавиш в редакторе.

Плагин умеет работать в режиме копирования файлов (вместо переименования).
Режим переименования/копирования переключается по F5 в редакторе.
Ограничение: копируются только файлы, копировать каталоги плагин не умеет.
